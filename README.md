# 项目名称
Kumquat是中国传统春节吉祥物金桔或金桔盆栽，有新年求财的意思。

# 项目简介
本项目是基于python的期货、股票历史数据分析及量化交易算法运行的工具软件。
支持软件用户自己在设置私有的交易准则，并且在历史数据上进行自动化模拟。

# 功能特点
## 基于永久免费的开源协议

## 同时支持期货及股票

## 简单直接的本地布置策略

## 支持自定义交易准则，记录在本地，完全属于用户个人的策略验算工具
